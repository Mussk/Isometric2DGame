# Isometric2DGame
This repository contains a Unity Project of Isometric 2D Game created as a part of a test assessment.

This projest is scopes on implementation of 4 main tasks:
- Unity Project Setup and GitHub Integration;
- Isometric Player Movement and Input System;
- AI State Machine for Enemy Behavior;
- Personal Specialization (Combat, Iventory, Dialogue Systems)

Controls:
WASD - Movement;
LMB - use primary weapon;
E - Pick up items:
R - On/Off automatic weapons;

After game start, on the left side of a player will be a pickable weapon, press E to pick it up, it will activate an automatic firing weapon. You can switch it ON/OFF by pressing R.

Used assets:
- Legacy Fantasy - Castle Prison
- Pixel Crawler - Cemetery
